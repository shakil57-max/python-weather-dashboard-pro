Python Weather Dashboard Pro

A simple and beginner-friendly desktop weather application built with Python, CustomTkinter and the Open-Meteo API.

This project was made to provide quick and clean weather information without any complicated setup. You can check the current weather, get a 7-day forecast, see hourly updates, sunrise and sunset times, and also view your previous searched cities.
The interface is clean, easy to understand, and supports both dark and light modes.


Features

1.Real-time weather information
2.7-day forecast
3.Hourly weather updates (2-hour interval)
4.Sunrise and sunset times
5.Optional voice-based search
6.Search history saved locally
7.Dark/Light mode
8.Works without any API key (uses Open-Meteo)

Installation
1. Clone the project
git clone https://github.com/shakil57-max/python-weather-dashboard-pro.git
cd python-weather-dashboard-pro


2. Install required libraries
pip install -r requirements.txt
pip install SpeechRecognition pyaudio
python main.py
